# Valorant React Website

A modern web application about Valorant, built with React.js.

## Features

- Latest Valorant news and updates
- Agent and map information
- Interactive guides and tips
- Responsive design

## Getting Started

1. **Clone the repository:**
    ```bash
    git clone https://github.com/your-username/valorant-react-website.git
    cd valorant-react-website
    ```

2. **Install dependencies:**
    ```bash
    npm install
    ```

3. **Start the development server:**
    ```bash
    npm start
    ```

4. Open [http://localhost:3000](http://localhost:3000) to view the app.

## Technologies Used

- [React.js](https://reactjs.org/)
- [React Router](https://reactrouter.com/)
- [Axios](https://axios-http.com/) (for API requests)
- CSS Modules / Styled Components
